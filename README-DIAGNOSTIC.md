# T-Stock Veren V10.5.3 TEST DIAGNOSTIC

Deze Android-testversie gebruikt een aparte app-ID en opent altijd eerst een volledig native diagnosescherm.

1. Open de app.
2. Druk op **T-Stock Veren starten**.
3. Sluit de hoofdapp opnieuw, open de app dan nogmaals.
4. De volledige Android-stacktrace staat nu in beeld en kan worden gekopieerd.

De versie initialiseert de scanner pas na een succesvolle login en gebruikt op het eerste scherm geen afbeelding via Compose.
